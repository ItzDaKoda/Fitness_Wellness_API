
# Fitness & Wellness Tracker API (MVP)

This project is a RESTful API for tracking workouts, fitness goals, and daily wellness logs.

## Tech Stack

- Node.js / Express
- SQLite with Sequelize ORM
- Jest + Supertest for testing

## Setup


npm install
npm run db:setup
npm run db:seed
npm run dev

postman API documentation: https://documenter.getpostman.com/view/49930747/2sB3dQtone
